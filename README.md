# ZEXXY Like Panel

## Backend
1. Navigate to `backend/`
2. Run `npm install`
3. Run `npm start` → Proxy will run on port 3000

## Frontend
1. Open `frontend/index.html` in browser
2. Enter UID and region → Click Send Likes
3. Real API call via backend proxy
